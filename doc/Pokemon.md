## 简中名称变更

2020 年 3 月 23 日更改，可能由于电子游戏审查制度（本人猜测），将罗马数字统一为阿拉伯数字，去掉了英文字母。

|  旧名称  |   新名称   |
| :------: | :--------: |
| 多边兽Ⅱ  | 多边兽２型 |
| 多边兽Ｚ | 多边兽乙型 |
|  谜拟Ｑ  |   谜拟丘   |

2020 年 10 月 20 日更改，和谐了部分名称。

|  旧名称  |  新名称  |
| :------: | :------: |
| 流氓熊猫 | 霸道熊猫 |
|  死神棺  |  迭失棺  |
|  偷儿狐  |  狡小狐  |
|  狐大盗  |  猾大狐  |
|  死神板  |  迭失板  |
|  毒电婴  |  电音婴  |

第九代和谐的宝可梦名称：

| 繁体名称 | 简体名称 |
| :------: | :------: |
| 仆斩将军 | 仆刀将军 |
|  厄鬼椪  |  厄诡椪  |

词库同时保留因为和谐造成的新旧名称，和谐前的名称来源于繁体译名。

## 异体字

简体繁体翻译存在异体字的区别，词库仅仅保留简体版本。火暴猴例外，因为两者都很常见，甚至火爆猴更常见。
还有一个例外是厄鬼椪/厄诡椪，这更像是和谐的产物。

|   简体   |   繁体   |
| :------: | :------: |
|  火暴猴  |  火爆猴  |
|  火暴兽  |  火爆兽  |
|  荧光鱼  |  萤光鱼  |
|  保姆虫  |  保母虫  |
|  车轮球  |  车轮毬  |
| 保姆曼波 | 保母曼波 |
|  穿着熊  |  穿著熊  |
